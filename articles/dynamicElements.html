Building Dynamic ELements with JavaScript

April 1st, 2019
Intro

Today we are going to talk about building dynamic elements in Vanilla JavaScript! I am actually pretty excited about this topic because it is something that is extremely useful, yet I don't feel like there are many resources on the web on how to do this. We want our web application to be dynamic so and we want to understand how to do this with JavaScript before jumping into a frontend framework like React, Angular Vue.js or using (no longer needed) jQuery. We are going to take 3 very practical elements that I build all the time in my current Dev role and walk through the whole process. We will create a button for quick demo purposes then create an ordered list, a select dropdown and a table.

This article might get a little lengthy but it is mostly code examples. With each element that we are building we will first build it with a hard coded array of json and then we will build the exact same thing but with a fetch method pulling data from mockAPI. If you are unfamiliar with the fetch method you might want to read my article on ES6 Fetch Method first.

Syntax

Let’s get started with the basic syntax for creating elements and then adding them to the DOM in a specific place. To create an element you will use the following method: document.createElement(). You can create any valid HTML elements like buttons, lists, selects, tables etc. Let’s talk about creating a button first. The element that you are creating is what is being passed into the method. You also want to store that newly created element in a variable so you can continue to use it. For example: const myButton = document.createElement(“button”) Now that your new button is in a variable ‘myButton’ you can chain more methods on it like adding a class or setting an attribute. We will go over that more in the remaining examples in this article.

So you have this button, but where are you going to put it? To add this new button to the DOM you need to append it to something with the following method: element.appendChild(). In the English dictionary the definition of append is: verb add (something) as an attachment or supplement. Let’s say you have a div with the ID of ‘myDiv’ and you want to nest your button to the div. The syntax would look like this:
const parentDiv = document.getElementById(“myDiv”) parentDiv.appendChild(‘myButton’)


We are going to take this same principal and apply throughout our examples of building dynamic lists, selects and tables. Are you excited? Thought so.. Get your text editor ready with the following files: index.html, style.css, list.js, select.js and table.js Be sure to link those files appropriately to your index.html file and then we are ready to go.

Dynamic Tables

Hopefully at this point you are getting the hang of dynamically creating elements and appending them to the parent element. We are now going to build a dynamic table which has more elements so can be slightly tricky but I am going to break it all down like the previous examples so let’s get started.

First take a look at a table on HTML. There are more nested elements that make up a table so mapping this out ahead of time is even more important. I did add some simple css to this table which you can grab from my GitHub but it is not necessary.

Now lets walk through step by step what we need to create, looking at the HTML too keep us on track.

  1. Create a <div> in your HTML
  2. Create a <button> in the HTML with an ID to display your table
  3. Head over to your JS file and get that <div> by the ID and store in a variable
  4. Create a <table> element and store in a variable
  5. Create a <thead> element and store in a variable
  6. Create <tr> element that will be nested in the <thead> and store in a variable
  7. Create <th> element that will be for your table heading names and store in a variable
    • You will need one for each table heading. In my example I will be creating 4
  8. Create <tbody> element that will be used later in the forEach loop
  9. Set the innerText of each of the 4 <th> that you created in #6
    • This is what the user will see. My example is; “Name”, “Value”, “Color” and “Qty”
  10. Append each of the <th> variables to the <tr>
  11. Append the <tr> to the <thead>
  12. Now we will need to loop through our array and for each fruit in the fruits array we want to:
    • Create <tr> element that will be nested in the <tbody> and store in a variable
    • Create <td> element that will be nested in the <tr> and store in a variable
    • Set the innerText of each of your <td> that you just created using dot notation
    • Append each of the <td> variables to the <tr>
    • Append the <tr> variable to the <tbody>
    • Close the forEach loop
  13. Now that you have created all the elements of the table you need to append the following:
    • <table>to the <div>
    • <thead> to the <table>
    • <tbody> to the <table>
  14. Add an event listener to your button to call this function that build the table
  15. That’s it!!!
