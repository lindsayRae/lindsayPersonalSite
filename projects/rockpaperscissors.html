Rock Paper Scissors

The project

Completed May 2018. This is a demonstration of the classic game: "Rock Paper Scissors". The user is playing versus the computer where the user selects rock, paper or scissors and the program will tell the user if they won or if the computer did.

Focus

I really wanted to focus on the logic for this project since I was just getting started with JavaScript and jQuery.

See it live!

Check out the code

What I learned

This was a great beginner project where I pulled together jQuery and some basic logic using if else statements, Math.random() functions and return statements. For the display I using Bootstrap utilizing the column grid system and purchased the images from themeforest.
